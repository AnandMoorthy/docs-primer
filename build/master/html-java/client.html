Java Driver¶

MongoDB Java Driver is the officially supported Java driver for MongoDB.

Procedure¶

1

Download the MongoDB Java Driver and the BSON library.¶

From the Java Driver Installation Page, follow the instructions to install the MongoDB Java Driver as well as to install the BSON library. The page provides details to install via a dependency management system or to download the jars directly from The Central Repository.

2

Connect to MongoDB.¶

Use com.mongodb.MongoClient class to connect to a running mongod instance. Access a specific database in MongoDB using the com.mongodb.client.MongoDatabase interface.

Include the following import statements.

import com.mongodb.MongoClient;
import com.mongodb.client.MongoDatabase;

Connect to a MongoDB instance running on the localhost on the default port 27017:

MongoClient mongoClient = new MongoClient();

To specify a different host and port, see the com.mongodb.MongoClient API page.

Once successfully connected, access the test database:

MongoDatabase db = mongoClient.getDatabase("test");
